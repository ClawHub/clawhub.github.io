深入了解Mysql【十六】主从复制延迟的解决-并行复制

发表于2019-11-21|更新于2023-09-26|mysql

MySQL主从复制有个很大的问题：延迟。
从节点复制包括两个部分:

  • IO线程读取binlog,写入到中继日志
  • SQL线程解析中继日志，执行SQL,进而和主节点数据同步

发现IO线程不是瓶颈，要在SQL线程上解决问题，所以提出了并行复制的概念：
只要再SQL线程上实现多线程，提升从节点处理数据的并发，那么数据同步进度就跟上来了。
