Hexo+Next之使用CODING企业版做博客持续集成CI

发表于2019-08-06|更新于2023-09-26|hexo
|阅读量:

如果有了持续集成操作，就可以在任何一台联网并且装有git的电脑上写文章了，不必每次手动部署。

尝试了好久才发现CODING企业版做博客持续集成是真的很方便，前提是简单的了解jenkins配置。
流程：

  1. 首先注册CODING企业版，创建一个项目
    如：blog-source,最好是私有的，因为涉及到密码子类的东西。

  2. 之后在项目左侧面板找到持续集成菜单
    点击Job列表，新建，并配置Jinkens。

  3. 上传博客源码到此项目中。

  4. 上传代码后会自动触发Job。

  5. 等待一会，查看博客。
